# Market Data Fetcher

A tool for fetching market data from various sources. This README provides installation and usage instructions to get you started quickly.

## Installation

To install the Market Data Fetcher, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/john901028-wq/TMF1-.git
   cd TMF1-
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

After installation, you can use the Market Data Fetcher by running:

```bash
python fetcher.py --source <data_source> --output <output_file>
```

### Parameters:
- `--source`: The source from which to fetch the market data. Supported sources include 'API1', 'API2', etc.
- `--output`: The name of the file where the fetched data will be saved.
